Consensia

A Bilkent University CS491 research project inspired by “LLM as a Judge”, supervised by Prof. Anıl Koyuncu. Our goal is to explore whether a large language model can serve as a trustworthy and explainable judge in the domain of software engineering.

Prototype & Approach

Our prototype allows users to create multiple AI personas with different software-engineering roles such as CTO, Developer, and Security Engineer. Each agent generates a response to a given question, while a Judge LLM analyzes all outputs, scoring them for consistency, fairness, and reasoning clarity. The judge’s consensus is then validated against human-tagged data.

System Concept

Multi-agent discussion leading to a final, explainable verdict generated by the Judge LLM, combining text-based reasoning and evidence from tools.

Architecture

Event-driven structure enabling real-time processing and scalability. Frontend in React, backend using Python, and automation via n8n.

LLM Core

Powered by Ollama, Deepseek, and Claude. Analysis and statistics handled with Python libraries, benchmarked against expert-labelled datasets.

Validation

Outputs are compared with real-world datasets to assess reasoning quality and alignment with human judgment.
